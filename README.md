# Statuses

Easily scan trough folders with multiple repos. Useful for multi repo projects.

## Installation


### Using Git
```bash
git clone https://github.com/your-username/statuses.git
cd statuses
chmod +x statuses
sudo mv statuses /usr/local/bin/statuses
#reload your shell
```


## Usage
```bash
statuses
statuses --help
statuses --version
statuses --simple
statuses -a
```
